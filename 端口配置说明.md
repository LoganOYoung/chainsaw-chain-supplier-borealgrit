# 端口配置说明

## ✅ 已统一到 3000 端口

项目现在**只使用 3000 端口**，配置已简化。

---

## 📍 网站访问地址

**唯一地址：http://localhost:3000/**

---

## 🚀 启动方式

### 方法 1：使用 npm 命令
```bash
npm run dev
```

### 方法 2：使用启动脚本
```bash
./start-dev.sh
```

### 方法 3：双击启动（macOS）
双击 `一键启动.command` 文件

---

## 📝 配置变更

### 已移除
- ❌ `dev:8080` 命令（已删除）
- ❌ `start:8080` 命令（已删除）
- ❌ 8080 端口相关配置

### 保留
- ✅ `dev` → 启动在 3000 端口
- ✅ `start` → 生产模式在 3000 端口
- ✅ `build` → 构建项目

---

## ⚠️ 注意事项

1. **只使用 3000 端口**：所有启动命令都使用 3000 端口
2. **如果 3000 端口被占用**：
   - 查找占用进程：`lsof -i:3000`
   - 停止进程：`kill -9 <PID>`
   - 或者修改 `package.json` 中的端口号

---

## 🔍 验证服务器状态

检查服务器是否运行：
```bash
lsof -i:3000
```

如果看到输出，说明服务器正在运行。
