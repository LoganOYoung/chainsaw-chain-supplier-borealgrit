# Google Analytics 连接指南

## 📊 Google Search Console vs Google Analytics

### Google Search Console（已连接）
- **用途**：SEO数据、搜索表现、索引状态
- **数据**：搜索查询、点击率、排名、索引页面数
- **重点**：搜索引擎优化

### Google Analytics（需要连接）
- **用途**：网站流量、用户行为、转化数据
- **数据**：访问量、页面浏览、用户停留时间、跳出率、转化
- **重点**：用户行为分析

**两者结合**：可以查看从Google搜索来的用户在你的网站上的行为

---

## 步骤 1：创建 Google Analytics 账号

### 1.1 访问 Google Analytics
1. 访问：https://analytics.google.com/
2. 使用你的 Google 账号登录（建议使用与 Search Console 相同的账号）

### 1.2 创建账号
1. 点击 **"开始测量"** 或 **"创建账号"**
2. 填写账号信息：
   - **账号名称**：例如 "Borealgrit Business"
   - **数据共享设置**：根据需要选择（可以全部勾选）
3. 点击 **"下一步"**

### 1.3 创建属性
1. **属性名称**：`borealgrit.com`
2. **报告时区**：选择你的时区（例如：America/New_York）
3. **货币**：USD（美元）
4. 点击 **"下一步"**

### 1.4 设置业务信息
1. **行业类别**：选择 "制造业" 或 "B2B"
2. **业务规模**：选择合适的大小
3. **使用目的**：选择你需要的功能
4. 点击 **"创建"**

### 1.5 接受服务条款
1. 阅读并接受 Google Analytics 服务条款
2. 点击 **"我接受"**

## 步骤 4：连接 Google Analytics 和 Search Console

### 4.1 在 Google Analytics 中连接
1. 登录 Google Analytics
2. 进入 **"管理"**（左下角齿轮图标）
3. 在"属性"列中，点击 **"Search Console 关联"**
4. 点击 **"关联 Search Console"**
5. 选择你的 Search Console 属性：`borealgrit.com`
6. 点击 **"关联"**

### 4.2 在 Search Console 中连接
1. 登录 Google Search Console
2. 进入 **"设置"**（左下角齿轮图标）
3. 点击 **"关联"**
4. 找到 Google Analytics，点击 **"关联"**
5. 选择你的 Google Analytics 属性
6. 点击 **"保存"**

## 步骤 5：查看详细数据

### 5.1 Google Search Console 中的详细数据

#### 性能报告（Performance）
1. 在左侧菜单中，点击 **"性能"**
2. 查看：
   - **总点击次数**：从Google搜索来的点击
   - **总展示次数**：你的网站在搜索结果中出现的次数
   - **平均点击率 (CTR)**：点击次数/展示次数
   - **平均排名**：你的网站在搜索结果中的平均位置

#### 查询（Queries）
- 查看用户搜索的关键词
- 哪些关键词带来了流量
- 哪些关键词排名好但点击率低

#### 页面（Pages）
- 哪些页面被搜索最多
- 哪些页面排名最好
- 哪些页面需要优化

#### 国家/地区（Countries）
- 流量来自哪些国家
- 不同国家的搜索表现

#### 设备（Devices）
- 移动端 vs 桌面端的搜索表现
- 不同设备的点击率

### 5.2 Google Analytics 中的详细数据

#### 实时报告（Real-time）
- 当前有多少人在浏览网站
- 他们正在看哪些页面
- 他们来自哪里

#### 受众群体（Audience）
- **用户数**：访问网站的用户数量
- **新用户 vs 回访用户**
- **地理位置**：用户来自哪些国家/城市
- **设备**：移动端、桌面端、平板
- **浏览器**：Chrome、Safari、Firefox等
- **年龄和性别**（如果数据足够）

#### 流量获取（Acquisition）
- **流量来源**：
  - Google搜索（有机搜索）
  - 直接访问
  - 推荐网站
  - 社交媒体
- **Google搜索数据**（连接Search Console后）：
  - 搜索查询
  - 点击率
  - 平均排名

#### 行为（Behavior）
- **网站内容**：
  - 哪些页面最受欢迎
  - 用户浏览路径
  - 退出页面
- **网站速度**：
  - 页面加载时间
  - 需要优化的页面

#### 转化（Conversions）
- **目标**：可以设置转化目标
  - 例如：提交询价表单
  - 下载PDF
  - 访问特定页面
- **电子商务**：如果有在线销售

### 5.3 连接后的综合数据

连接 Search Console 和 Analytics 后，可以在 Analytics 中看到：

1. **搜索查询报告**
   - 路径：获取 > Search Console > 查询
   - 查看：哪些搜索词带来了流量，用户行为如何

2. **着陆页报告**
   - 路径：获取 > Search Console > 着陆页
   - 查看：从搜索来的用户访问了哪些页面

3. **国家/地区报告**
   - 路径：获取 > Search Console > 国家/地区
   - 查看：不同国家的搜索表现和用户行为

4. **设备报告**
   - 路径：获取 > Search Console > 设备
   - 查看：不同设备的搜索表现和用户行为

## 步骤 6：设置转化目标（可选但推荐）

### 6.1 创建目标
1. 在 Google Analytics 中，进入 **"管理"** > **"目标"**
2. 点击 **"创建目标"**
3. 选择目标类型：
   - **目标网址**：例如，询价成功页面
   - **事件**：例如，点击下载按钮
   - **持续时间**：例如，访问超过5分钟
   - **页面/屏幕数**：例如，浏览超过3页

### 6.2 推荐的转化目标
对于你的网站，建议设置：
1. **提交询价表单**
   - 类型：目标网址
   - 网址：`/request-quote` + 成功参数
2. **下载PDF资源**
   - 类型：事件
   - 事件：点击下载按钮
3. **访问产品页面**
   - 类型：目标网址
   - 网址：`/products/*`
4. **访问询价页面**
   - 类型：目标网址
   - 网址：`/request-quote`

## 数据解读指南

### Search Console 关键指标

1. **点击次数**
   - 从Google搜索来的实际访问
   - 目标：持续增长

2. **展示次数**
   - 网站在搜索结果中出现的次数
   - 目标：增加展示 = 提高排名

3. **点击率 (CTR)**
   - 点击次数 / 展示次数
   - 目标：> 3%（B2B网站）
   - 如果CTR低：优化标题和描述

4. **平均排名**
   - 在搜索结果中的平均位置
   - 目标：< 10（第一页）
   - 排名1-3：最佳
   - 排名4-10：良好
   - 排名>10：需要优化

### Analytics 关键指标

1. **用户数**
   - 访问网站的唯一用户
   - 目标：持续增长

2. **会话数**
   - 用户访问次数
   - 目标：增加回访

3. **跳出率**
   - 只看一页就离开的用户比例
   - 目标：< 50%（B2B网站）
   - 如果高：优化内容、改善用户体验

4. **平均会话时长**
   - 用户在网站停留的平均时间
   - 目标：> 2分钟
   - 如果低：内容需要更有吸引力

5. **页面/会话**
   - 每次访问浏览的页面数
   - 目标：> 2页
   - 如果低：改善内部链接、导航

## 常见问题

### Q: 数据什么时候开始显示？
A: 
- Search Console：提交sitemap后几天内开始显示数据
- Analytics：安装代码后立即开始收集数据，但报告需要24-48小时

### Q: 为什么Search Console和Analytics的数据不一致？
A: 
- Search Console：只统计从Google搜索来的流量
- Analytics：统计所有流量来源
- 这是正常的，两者互补

### Q: 如何提高搜索排名？
A: 
- 优化页面标题和描述
- 改善内容质量
- 增加内部链接
- 获取高质量外链
- 提高页面加载速度
