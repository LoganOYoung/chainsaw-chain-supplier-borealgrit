# 域名邮箱配置说明

## 🎯 为什么使用域名邮箱？

使用 `@borealgrit.com` 域名邮箱的优势：

✅ **专业形象** - 邮件来自公司域名，提升品牌信任度  
✅ **品牌一致性** - 与网站域名一致，统一品牌形象  
✅ **更好的送达率** - 域名邮箱通常有更好的邮件送达率  
✅ **便于管理** - 可以创建专门的邮箱（如 `sales@borealgrit.com`）

---

## 📧 域名邮箱服务商选择

### 方案 1：Google Workspace（推荐）⭐

**价格：** $6/月起/用户  
**优点：**
- ✅ 稳定可靠
- ✅ 与 Gmail 相同的界面
- ✅ 支持 SMTP
- ✅ 专业的企业邮箱功能

**配置步骤：**
1. 注册 Google Workspace：https://workspace.google.com/
2. 验证域名所有权
3. 创建邮箱账号（如 `sales@borealgrit.com`）
4. 在 EmailJS 中使用 SMTP 配置：
   - Host: `smtp.gmail.com`
   - Port: `587` (TLS)
   - Username: `sales@borealgrit.com`
   - Password: 邮箱密码或应用专用密码

---

### 方案 2：Microsoft 365（Office 365）

**价格：** $6/月起/用户  
**优点：**
- ✅ 与 Outlook 集成
- ✅ 企业级功能
- ✅ 支持 SMTP

**配置步骤：**
1. 注册 Microsoft 365：https://www.microsoft.com/microsoft-365
2. 添加域名并验证
3. 创建邮箱账号
4. 在 EmailJS 中使用 SMTP 配置：
   - Host: `smtp.office365.com`
   - Port: `587` (STARTTLS)
   - Username: `sales@borealgrit.com`
   - Password: 邮箱密码

---

**价格：** 免费版支持 5 个用户  
**优点：**
- ✅ 免费版可用

- ✅ 支持 SMTP

4. 在 EmailJS 中使用 SMTP 配置：
   - Host: `smtp.zoho.com`

   - Username: `sales@borealgrit.com`
   - Password: 邮箱密码

---

### 方案 4：使用现有邮箱服务

如果你已经有域名邮箱服务（如通过 GoDaddy、Namecheap 等购买的邮箱），可以直接使用：

1. 查找你的邮箱服务商的 SMTP 设置
2. 常见 SMTP 设置：
   - **GoDaddy**: `smtpout.secureserver.net` (Port: 465)
   - **Namecheap**: `mail.privateemail.com` (Port: 587)
   - **其他**: 查看你的邮箱服务商文档

## 🔧 EmailJS SMTP 配置示例

### Google Workspace 配置

```
Service Name: Borealgrit Domain Email
Host: smtp.gmail.com
Port: 587
Username: sales@borealgrit.com
Password: [你的邮箱密码或应用专用密码]
Secure: TLS
```

### Microsoft 365 配置

```
Service Name: Borealgrit Domain Email
Host: smtp.office365.com
Port: 587
Username: sales@borealgrit.com
Password: [你的邮箱密码]
Secure: STARTTLS
```

## 🔐 安全设置

### Google Workspace 应用专用密码

如果启用了两步验证，需要使用应用专用密码：

1. 访问：https://myaccount.google.com/apppasswords
2. 创建新的应用专用密码
3. 在 EmailJS 中使用这个密码（不是邮箱密码）

### Microsoft 365 SMTP AUTH

确保启用了 SMTP AUTH：

1. 登录 Microsoft 365 Admin Center
2. 进入 **Settings** → **Mail**
3. 启用 **SMTP AUTH**
