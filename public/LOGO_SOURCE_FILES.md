# BOREALGRIT Logo 源文件说明

## 文件列表

所有logo源文件均为SVG格式，可直接下载使用。

### 完整Logo（带品牌名称）

1. **logo-bg-white.svg** - 白色版本（用于深色背景）
   - 尺寸：200×60px
   - 用途：Header、Footer
   - 颜色：白色

2. **logo-bg.svg** - 绿色版本（用于浅色背景）
   - 尺寸：200×60px
   - 用途：浅色页面背景
   - 颜色：#547950（品牌绿色）

### 图标版本（仅BG字母）

3. **logo-bg-icon-white.svg** - 白色图标版本
   - 尺寸：60×60px
   - 用途：Favicon、小尺寸应用
   - 颜色：白色

4. **logo-bg-icon.svg** - 绿色图标版本
   - 尺寸：60×60px
   - 用途：Favicon、小尺寸应用
   - 颜色：#547950（品牌绿色）

## 设计说明

- **创意来源**：BG两个字母（BOREALGRIT的缩写）
- **设计风格**：工业链条风格，字母B和G通过链条连接环连接
- **字体**：Oswald（品牌名称部分）
- **颜色**：品牌绿色 #547950

所有文件位于 `/public/` 目录下，可通过以下方式下载：

   - 访问：https://github.com/LoganOYoung/chainsaw-chain-supplier-borealgrit
   - 进入 `public/` 目录
   - 下载所需的SVG文件

2. **从网站直接下载**：
   - 访问：https://borealgrit.com/logo-bg-white.svg
   - 访问：https://borealgrit.com/logo-bg.svg
   - 访问：https://borealgrit.com/logo-bg-icon-white.svg
   - 访问：https://borealgrit.com/logo-bg-icon.svg

3. **本地文件系统**：
   - 文件路径：`/Users/ouyangheng/coding/chainsaw-chain-supplier-nextjs/public/`

## 使用建议

- **Header/Footer**：使用 `logo-bg-white.svg`
- **浅色背景页面**：使用 `logo-bg.svg`
- **Favicon**：使用 `logo-bg-icon-white.svg` 或 `logo-bg-icon.svg`
- **打印材料**：使用 `logo-bg.svg`（绿色版本）

## 编辑说明

所有SVG文件可直接用以下工具编辑：
- Adobe Illustrator
- Inkscape（免费）
- Figma
- 任何支持SVG的代码编辑器

SVG是矢量格式，可无损缩放至任意尺寸。
